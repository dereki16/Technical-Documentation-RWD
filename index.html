Starting Out

HTML or HyperText Markup Language is the standard markup language for documents designed to be displayed in a web browser.

        
<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>

  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Diving into HTML...

  • The <!DOCTYPE html> declaration defines that this document is an HTML5 document.
  • The <html> element is the root element of an HTML page.
  • The <head> element contains meta information about the HTML page.
  • The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab).
  • The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • The <h1> element defines a large heading.
  • The <p> element defines a paragraph.
Introduction

After having filled out the bare essentials, you'll want to properly label the elements and add a few things as well. 

        
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="styles.css" />
    <title>Page Title</title>
  </head>

  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Make sure to set the html to english language.

Add a meta tag with a charset equal to UTF-8 because it's able to represent all of the printable ASCII characters, as well as the non-printable characters

Add another meta tag for the viewport so your page will stay consistent despite devide accessed on.

Body

The body is where you'll make up the bulk of the page. This is where you'll want to add headings of the topics you'll be discussing, as well as paragraphs or media.

You can enable headings and paragraphs. 

        
    <h1>Title of Page - most important heading</h1>
    <h2>Topic of Importance</h2>
    <h3>Subtopic</h3>
    // this goes on until heading 6 - the least important heading

    <p>This is where you'd write your paragraph. Talk about the things that interest you, or describe the heading you listed above.</p>

In order to add a picture to your page, add an img tag and give it a source - such as a url.

          
    <img src="https://images.theconversation.com/files/377569/original/file-20210107-17-q20ja9.jpg?ixlib=rb-1.1.0&rect=278%2C340%2C4644%2C3098&q=45&auto=format&w=926&fit=clip">
    <img src="https://www.dogville-daycare.com/assets/images/cat_kity1.png">

Then you'll end up with a couple of images to display on your page!

Setting Up Elements

This is where we will set up elements with appropriate id and class names. We will use ids for individual elements we want to modify. Class names for elements that will repeat.

Let's say for example, we want to change attributes for the second header, key words and images. 

        
    <h1>Animals Domesticated</h1>
    <h2>Dogs</h2>
    <h3>How dogs provide for their humans</h3>

    <p>Dogs are man's best friend. They are <strong>loyal, kind, and affectionate.</strong></p>
    <img id="dog-image" src="https://images.theconversation.com/files/377569/original/file-20210107-17-q20ja9.jpg?ixlib=rb-1.1.0&rect=278%2C340%2C4644%2C3098&q=45&auto=format&w=926&fit=clip">
    <div class="divider"></div>

    <h2 id="cat-title">Cats</h2>
    <h3>How cats live as kings</h3>

    <p>Cats are also pretty cute. As seen in the book, <span class="emphasized">Awesome Photos of Cats.</span></p>
    <img id="cat-image" src="https://www.dogville-daycare.com/assets/images/cat_kity1.png">
    <div class="divider"></div>
CSS

Sometimes you might want to give the same properties to multiple elements, but have different properties for other attributes.

          
// gives padding to the top and bottom of the page
body {
  padding: 10 0;
}
// changes color of all h2 elements
h2 {  
  color: #0000DD;
}
// changes color of the element with id="cat-title"
#cat-title {
  color: red;
}
// italicizes and underlines 
.emphasized {
  font-style: italic;
  text-decoration: underline;
}

This is where you might want to give the elements you want to keep similar the same class, or simply list out all elements with the same type.

When you want to specify differences, give each element a different id. 

  
// centers image and provides a 
img { 
  border
  margin: auto;
  display: block;
  border: 10px solid black;
  max-width: 50%;
} 
// specifies border radius and gives space between image and border
#dog-image {
  border-radius: 4px;
  padding: 5px;
}
// gives larger border-radius
#cat-image {
  border-radius: 40px;
}
// provides a dividing line in betewen content
.divider {
  margin-top: 40px;
  border: 2px solid black;
  border-radius: 40%;
}
